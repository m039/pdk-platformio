# Padauk Microcontroller: development platform for [PlatformIO](http://platformio.org)

# Usage

1. [Install PlatformIO](http://platformio.org)
2. Create PlatformIO project and configure a platform option in [platformio.ini](http://docs.platformio.org/page/projectconf.html) file:

```ini
[env:development]
platform = https://github.com/m039/pdk-platformio.git
board = ...
...
```

# Examples

Examples are in [pdk-playground](https://github.com/m039/pdk-playground/tree/main) repository.
